About
Originally written to help visualize the algorithm during my summer internship at Preface, this python program uses the random library to generate random coordinates on a plot and Matplotlib to visualize the data. No other libraries were used.

K-means Clustering is a type of machine learning algorithm which attempts to group data into K clusters (K is usually determined by methods such as the Elbow Method) using unsupervised learning. The demo video shows 1000 points being randomly placed on a 100 by 100 coordinate plane and being grouped into 3 clusters.

Explaining the Algorithm

    Convergence
    Since the centroids stopped moving, we have reached convergence. The three clusters can be seen on the number line.

    The same logic and algorithm was used to write the 2-dimensional version. The number of clusters K used in the example was just an arbitrary number. As mentioned above, K is usually chosen using strategies like the elbow method which takes the sum of squared distances from each data point to its centroid to get the Within Cluster Sums of Squares (WSS). A plot of WSS vs number of clusters K can be made and usually the ideal K to select is the point in the graph which resembles an "elbow".

    Because K-means Clustering identifies clusters within data that may not have any obvious labels or relation to each other, it can be useful in finding hidden groups in complicated data.
