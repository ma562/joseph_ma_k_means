About
Originally written to help visualize the algorithm during my summer internship at Preface, this python program uses the random library to generate random coordinates on a plot and Matplotlib to visualize the data. No other libraries were used.

K-means Clustering is a type of machine learning algorithm which attempts to group data into clusters using unsupervised learning. The demo video shows 1000 points being randomly placed on a 100 by 100 coordinate plane and being grouped into 3 clusters.

A short explanation
The program is written such that 12 points are randomly generated along an interval between 0 and 100. The goal of the K-means Clustering algorithm is to group these points into 3 clusters without any information except their locations. The locations are as follows:
  • Point A: 0
  • Point B: 1
  • Point C: 14
  • Point D: 24
  • Point E: 25
  • Point F: 29
  • Point G: 50
  • Point H: 58
  • Point I: 61
  • Point J: 66
  • Point K: 81
  • Point L: 93
